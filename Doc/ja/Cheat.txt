======================
= チート機能に付いて =
======================

ﾁｰﾄ支援         コードサーチする際に使用します。
                ※チート支援は，そのウインドウがアクティブな時はエミュレーション
                  をポーズします。メインウインドウがアクティブになるとエミュレー
                  ションのポーズを解除します。

ﾁｰﾄｺｰﾄﾞ         コードをロードしたり編集したりします。
                ※0.50からチート支援ウインドウが開いている時にもダイアログが出せ
                  るようになりました。

ﾁｰﾄｺｰﾄﾞ有効     全てのコードを有効化します。
ﾁｰﾄｺｰﾄﾞ無効     全てのコードを無効化します。

ｹﾞｰﾑｼﾞｰﾆｰ       ゲームジーニーのコードファイルを開きます。
                仕組み的に有効／無効を切り替えられません。無効化したい場合は再度
                イメージを開き直してください。

==============
= チート支援 =
==============

検索結果        検索結果を表示しています。ダブルクリックでその位置のアドレスをデ
                ータ書き換えのアドレスに送ります。右クリックするとメニューを開き
                ます。

基数            １０進数か１６進数を選択します。

データ長        サーチや比較のデータ長を選択します。

領域            データの検索領域を選択します。

-検索-

開始            検索を新規に開始します。
更新            今回データを更新します。
Undo            直前のデータを１度だけUndo出来ます。

Old = Now       前回と今回のデータが同じデータを検索します。
Old != Now      前回と今回のデータが違うデータを検索します。
Old < Now       前回より今回のデータが大きいデータを検索します。
Old > Now       前回より今回のデータが小さいデータを検索します。
Old <= Now      前回より今回のデータが大きいか同じデータを検索します。
Old >= Now      前回より今回のデータが小さいか同じデータを検索します。

検索            データに設定されたデータを検索します。

-データ書き換え-

書き換え        アドレスとデータに設定された場所に書き込みます。ある程度絞り込み
                出来た時に試す事が出来ます。

================
= チートコード =
================

チートコードリストの左側に出ているアイコンは，現在のチートコードの状態を示してい
ます。

左上，右下が各々チートコードの有効／無効状態，チートキーの有効／無効状態を表し，
緑が有効，赤が無効となっています。要するにチートキーで変更したく無いチートコード
は右下を赤にしておく良いと言う事です。

選択してスペースバーか，アイコンをクリックしていくと状態が変化します。

また，チートコードリストをダブルクリックすると編集ダイアログを出す事が出来ます。

セーブ時はチートコードの状態もセーブする様にしてあります。

========================
= チートコードの書き方 =
========================

セパレータは，'-'，タブ，スペース等で区切って下さい。

行の先頭文字が';'の場合はコメントとみなします。

行の先頭が'#'で，続く数値でチートコードの初期状態を設定する事が出来ます。なお，
チートコードとの間には必ずセパレータを置いて下さい。

#0      チート無効／チートキー有効
#1      チート有効／チートキー有効
#2      チート無効／チートキー無効
#3      チート有効／チートキー無効

となっています。これら以外は無視されます。

※コード入力時も初期状態を設定出来ます。設定しない場合はコード有効／チートキー
  有効状態(アイコンが共に緑)の状態になります。

例）
; スパルタンＸ
#1 04A6-01-30           体力減らない(クリア時切り)
#1 0390-04-00000002     タイム減らない(クリア時切り)
#3 0000-01-FF           ダミーです

==============
= コード説明 =
==============

AAAA-BC-DD      EE...

AAAA            アドレス

B               タイプ
                ０：常に
                １：１回だけ
                ２：データより大きい時
                ３：データより小さい時

C               長さ
                １：１バイト
                ２：２バイト
                ３：３バイト
                ４：４バイト

DD              データ
                バイト数＊２桁並びます

EE...           コメント
                終端含め255バイトまで。

既存データを使えるように，NNNesterJのチートコード（テキストファイルからのみ）も
読めるようにしてありますが，全てが上手くいく保証はありません。
※コード入力ではNNNesterJのコードは使用できません。

===========================
= NNN用のコードの変換方法 =
===========================

全てに当てはまるわけではありませんが，NNNの5桁コードは以下の様に変換すると大体の
コードが使えると思います。

例)
10502-1-FF	→	6502-01-FF

要するに5桁ある場合に下4桁に$6000(16進数で)を足して続くハイフンの数値に0を付加す
るだけです。

ウインドウズに標準の電卓を関数モードにすると16進数計算が出来ますので利用してみて
下さい。何気にコピペも使えますので便利です。
